# Trello Clone with Drag and Drop

[![Netology AHJ DnD](https://github.com/Milka79rus/ahj-trello/actions/workflows/web.yml/badge.svg)](https://github.com/Milka79rus/ahj-trello/actions/workflows/web.yml)

Интерактивный клонированный интерфейс доски управления задачами Trello с продвинутой кастомной реализацией Drag and Drop (перетаскивание элементов мышью). 

##  Особенности и функционал

* **Полноценный Drag and Drop (DnD):** Перемещение карточек как внутри одной колонки, так и между тремя фиксированными колонками (TODO, IN PROGRESS, DONE).
* **Точный захват мыши:** Внешний вид курсора меняется на `grabbing`. Карточка удерживается ровно в той точке, за которую её изначально схватили (не привязывается к центру).
* **Интерактивный Placeholder:** При перетаскивании динамически рассчитывается высота карточки и под неё выделяется пустое серое место. Карточка встает "до" или "после" соседнего элемента в зависимости от положения курсора мыши.
* **Полноценный LocalStorage:** Все состояние доски полностью сохраняется в браузере. При обновлении страницы или перезапуске все созданные карточки и их порядок остаются неизменными.
* **Управление задачами:** Возможность добавлять новые текстовые карточки через форму разворачивания и удалять их при клике на крестик, появляющийся при наведении.

##  Стек технологий

* **JavaScript (ES6+)** — ООП архитектура, модули
* **Yarn** — управление зависимостями (в проекте отсутствует `package-lock.json`)
* **Webpack 5** — сборка стилей, картинок и JS (раздельные конфигурации для dev/prod)
* **ESLint** — автоматический контроль качества и форматирования кода
* **CSS Grid / Flexbox** — адаптивная верстка интерфейса доски

##  Развертывание и команды

### Установка зависимостей
```bash
yarn install
```

### Запуск проекта в режиме разработки
```bash
yarn start
```

### Сборка production-версии проекта
```bash
yarn build
```

### Проверка кода линтером
```bash
yarn lint
```
